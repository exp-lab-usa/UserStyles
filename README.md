# UserStyles

Just a collection of some website CSS styles edited with Stylus. Some are modified forks, some are original, most are works in progress.

Created and tested on MacOS, using Wavebox, a Chromium-based desktop browser. Not verified with any other OS, browser, or platform.

***Open to contribution, suggestion, and request.***
